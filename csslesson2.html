1 What is a div element?

The element is like a bucket that groups sections of a webpage together.

2 What is one way you can identify or "target" a div element?

A div can be identified or “targeted” by calling a class in CSS, it will target every div with the same class.

3 Why would you use a div element?

The purpose of it is to group content so it can be easily styled using the class or id attributes

4 What is a border property?

The border property allows you to style the border around an element.

5 What are some of te different border properties?

  • Border width
  • Border color
  • Border sides
  • Rounded borders

6 What is the difference between margins and padding

A margin is the spacing around an element from everything around it. A padding is the spacing content has from the element border.
